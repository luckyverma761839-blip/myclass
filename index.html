Project Title: To-Do List Web Application

Description: Build a simple To-Do List app using HTML, CSS, and JavaScript. Users can add, delete, and mark tasks as completed. The app stores tasks in the browser's local storage so they persist after refreshing the page.

  • HTML: Structure the task input, list, and buttons.
  • CSS: Style the list, completed tasks, and layout.
  • JavaScript: Handle adding, deleting, marking tasks, and saving to local storage.
